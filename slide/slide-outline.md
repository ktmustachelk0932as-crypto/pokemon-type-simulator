## 追加資料: 相性計算ロジック
**タイプ相性の計算方法**

```javascript
// 複数タイプの場合: 各相性を乗算
const multiplier = selectedDefenseTypes.reduce((acc, defType) => {
  const m = CHART[attackType]?.[defType] ?? 1.0;
  return acc * m;
}, 1.0);
```

**例: ほのお×くさタイプに対するみず技**
- ほのおタイプ: みず技は効果バツグン (×2.0)
- くさタイプ: みず技は効果いまひとつ (×0.5)
- 最終結果: 2.0 × 0.5 = 1.0 (等倍)
